## cz-icon-lib 是什么

cz-icon-lib 是一个纯 css 实现的拥有各式各样的图标库

## 介绍

有了这个图标库，在实际项目中在也不用去使用图片和字体文件了，这样会大大减少我们项目的体积，以此也可以提高性能。
